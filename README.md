Meeting Room

A simple meeting room management / reservation system that allows users to manage and organize meeting rooms efficiently.
The project aims to demonstrate basic backend and/or full-stack development concepts such as resource management, booking logic, and API usage.

📌 Features

🏢 Meeting room listing

📅 Create meeting reservations

❌ Prevent overlapping reservations

🔍 View existing bookings

⚡ Simple and clean project structure

⚙️ Installation

Clone the repository:

git clone https://github.com/berkay-ucar/meetingroom.git

Navigate to the project directory:

cd meetingroom

Install dependencies:

npm install

Run the project:

npm start
🚀 Usage

Start the application.

Create or view meeting rooms.

Book a room for a specific time.

The system checks for overlapping reservations.
